# WhatsApp Clone (HTML & CSS)

Este proyecto es un **clon visual simple de WhatsApp Web**, desarrollado únicamente con **HTML y CSS**, con el objetivo de practicar maquetación, Flexbox y diseño de interfaces de chat.

**No es una aplicación funcional**: no incluye JavaScript, backend ni envío real de mensajes.

## Objetivo del proyecto

- Practicar **layout con Flexbox**
- Replicar la estructura visual de una app de mensajería
- Trabajar jerarquía visual y diseño UI
- Entender la organización de un chat (sidebar, header, mensajes, input)

## Tecnologías usadas

- **HTML**
- **CSS**
  - Flexbox
  - Grid (en algunas partes)
  - Variables CSS
